inline

Species other types that can be substituted for T when calling a function expecting T but T is not being passsed. Only available for classes derived from T. Every type is always compatible with itself and should not be specified here. Not calling this means that only T objects will be accepted for things that want a T. There is no automatic determination of inherited types by this library because I cannot figure out how. It's VERY important that any type marked as having this type as a parent be marked as being a compatible type.
