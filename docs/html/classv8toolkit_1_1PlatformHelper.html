std::shared_ptr< IsolateHelper > v8toolkit::PlatformHelper::create_isolate ( )
static

Creates a new IsolateHelper wrapping a new v8::Isolate instance. Each isolate is completely separate from all the others and each isolate can be used simultaneously across threads (but only 1 thread per isolate at a time) An IsolateHelper will remain as long as the caller has a shared_ptr to the IsolateHelper or any ContextHelpers created from the IsolateHelper still exist. Once neither of those things is the case, the IsolateHelper will be automatically destroyed. If any threads are still running when this happens, the results are undefined.
