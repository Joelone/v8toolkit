Detailed Description

template<class T>
class v8toolkit::V8ClassWrapper< T >

Provides a mechanism for creating javascript-ready objects from an arbitrary C++ class Can provide a JS constructor method or wrap objects created in another c++ function

Species other types that can be substituted for T when calling a function expecting T but T is not being passsed. Only available for classes derived from T. T is always compatible and should not be specified here. Not calling this means that only T objects will be accepted for things that want a T. There is no automatic determination of inherited types by this library because I cannot figure out how.
