Detailed Description

template<int depth, typename FUNCTION_TYPE, typename RET, typename HEAD, typename... TAIL>
struct ParameterBuilder< depth, FUNCTION_TYPE, std::function< RET(HEAD, TAIL...)> >

specialization that strips off the first remaining parameter off the function type, stores that and then inherits from another instance that either strips the next one off, or if none remaining, actually calls the function The function type is specified twice because the first is actually used by the final specialization to hold the function type while the second one has its input parameter list stripped off one at a time to determine when the inheritance chain ends
